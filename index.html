Lesson 1, HTML and the Web

HTML is the language of the web,it stands for Hyper Text Markup language. The majority of the web pages that we view are based in HTML. In order to access these HTML documents we must what we call the Web, which is a massive agregate of computeres that communicate with one another. Because the web has such a wealth of data we employ the use of a search engine/browser such as Google Chrome to assist us in finding the files that are relevant to what we are looking for.

HTML continued

The HTML documents of the web are made up primarily with two items, which are: Elements, and Tags. Elements hold all of the content and how they are displayed is determined using tags. For example I have made the word Bold bold with a bold tag. There are two types of elements which are: Block and Inline. Inline elements usually happen on the same line of text as they are written, and block elements form invisible boxes arount the content they enclose.

Lesson 2, The structure of HTML

HTML as I stated earlier is made up of elements and tags. By using developer tools via google chrome you can inspect a page and the elements within it. When looking at a page with developer tools you can see the entire structure of a page and all of its elements.

When you break down the structure of an HTML page you can see that it is made up of many boxes. Some of these boxes are even nested within other boxes creating a tree like structure.
